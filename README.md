# Healthcare Chatbot on AWS

This project implements a basic healthcare chatbot using:

* Amazon Lex
* AWS Lambda (Python)
* DynamoDB
* Terraform IaC

## Folder Structure

* /terraform → AWS infra
* /lambda → Lambda chatbot logic
* /frontend → Simple UI
* /docs → Docs + diagrams
